# RAG System Frontend

React Ionic frontend for the RAG (Retrieval-Augmented Generation) system.

## Features

- **Modern UI**: Built with Ionic React components
- **Chat Interface**: Real-time chat with the RAG system
- **Document Management**: Upload, view, and delete documents
- **API Settings**: Configure API keys for multiple LLM providers
- **Responsive Design**: Works on desktop and mobile devices

## Installation

```bash
cd frontend
npm install
```

## Configuration

The `.env` file is already configured with `REACT_APP_API_URL=http://localhost:8000`

## Available Scripts

- `npm start` - Runs the app in development mode
- `npm run build` - Builds the app for production
- `npm test` - Launches the test runner
